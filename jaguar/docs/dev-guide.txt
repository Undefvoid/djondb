- tener cuidado con los char*, cuando se quiere copiar debe cerrarse el char* con 0 asi:

    char* entityData = (char*) malloc(sizeof(char) * (len+1));
    memset(entityData, 0, len);
    strncpy(entityData, data, len);
    entityData[len] = 0;

- Ojo con el strtok, este deja un apuntador a la cadena actual, si se libera la cadena no hay una copia del dato.

char* s = "esto es una prueba";
char* n = strtok(s, " "); // n apunta a el espacio dentro de s, si s el freed entonces n apunta a un sitio que no existe

- Los char* deben copiarse usando la longitud + 1, si no se hace asi el null terminated no se copia y genera errores, ejemplo:

char* source;
....
char* destination = (char*)malloc(strlen(source) + 1); // --> el +1 es importante
strcpy(destination, source);
